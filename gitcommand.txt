删除分支test,不能删除当前所在的分支

git branch -D test (D强制删除)

合并分支到master：
1. dev 分支 
git add .
git commit -m xxxxxx
2. 切换分支
git checkout master
3.合并分支
git merge dev


当git merge(git checkout master,只能在主分支合并)，合并解决冲突：
1.保留当前分支的代码，忽略其他分支
git merge --abort 
2. 手动修改，删除特殊符号
git add .
git commit
git commit -m 解决冲突之后的代码
